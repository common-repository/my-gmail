== Description ==

My GMail is a simple dashboard widget which display all your unread mails from your GMail account.
Only works over HTTPS !

== Installation ==

1. Upload `my-gmail.zip` content to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
